Documentation

  • getTankId()

    Permet de connaître l'identifiant de votre tank.

    Aucun paramètre
    Retourne: variable de type TankID.

  • getEnemyTankId()

    Permet de connaître l'identifiant du tank ennemie.

    Aucun paramètre
    Retourne: variable de type TankID.

  • getCellDistance(cell_1, cell_2)

    Permet de connaître la distance entre deux cellules de la carte.

    Paramètres:

    • cell_1: type NumCell

    • cell_2: type NumCell


    Retourne: valeur numérique.

    Indice: imaginez la distance entre deux points dans un repère orthonormé.

  • getCellFromXY(x, y)

    Permet de connaître le numéro d'une cellule à partir de x et y.

    Paramètres:

    • x: valeur numérique

    • y: valeur numérique


    Retourne: variable de type NumCell.

  • getCellPosX(cell)

    Permet de connaître l'abscisse X de la cellule cell.

    Paramètres:

    • cell: type NumCell


    Retourne: valeur numérique.

  • getCellPosY(cell)

    Permet de connaître l'ordonnée Y de la cellule cell.

    Paramètres:

    • cell: type NumCell


    Retourne: valeur numérique.

  • getPosition(id)

    Permet de connaître le numéro de la cellule sur laquelle se trouve le tank qui a pour identifiant id.

    Paramètres:

    • id: type TankID


    Retourne: variable de type NumCell.

  • getLife(id)

    Permet de connaître la vie restante du tank qui a pour identifiant id

    Paramètres:

    • id: type TankID


    Retourne: valeur numérique.

  • getPA(id)

    Permet de connaître le nombre de Point d'Action actuel (ou restant) du tank d'identifiant id.

    Paramètres:

    • id: type TankID


    Retourne: valeur numérique.

  • getRange(id)

    Permet de connaître la portée du tank d'identifiant id.

    Paramètres:

    • id: type TankID


    Retourne: valeur numérique.

  • moveTank(cell [, n])

    Fonction permettant de déplacer son tank vers une cellule NumCell.
    Si celle-ci n'est pas à sa portée, le tank se déplace vers une cellule proche de celle désiré.

    Action réitérable en fonction des paramètres courant.

    Paramètres:

    • cell: type NumCell

    • n: valeur numérique - OPTIONNEL

    Par défaut le nombre de points d'action (PA) utilisé est de 2 PA.
    Si n est renseigné, alors la fonction utilise n PA pour bouger.

    Aucune valeur de retour.

  • shoot()

    Fonction permettant de tirer sur le tank ennemi.

    Aucun paramètre
    Action réitérable en fonction des paramètres courant.
    Aucune valeur de retour.
